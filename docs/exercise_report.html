問題設定

テーマ

本物のアート（Authentic Art）とAI生成アート（AI-Generated Art）を分類する教師あり学習モデルを構築し、SHAPを用いてモデルの解釈を行う。

目的

  • WikiArtデータセットの印象派絵画（本物）とStable-Diffusionで生成された画像（フェイク）を分類する
  • ランダムフォレスト分類器を用いて高精度な分類モデルを構築する
  • SHAPを用いて、モデルがどの特徴量に基づいて分類を行っているかを解釈する

SHAP Summary Plotの解釈

図2: SHAP Summary Plot

主な発見

  • テクスチャ特徴量が最も重要: LBP値の標準偏差（texture_lbp_std）が最も高い重要度を示しています。本物の絵画は筆触や技法により多様なテクスチャパターンを持つ傾向があります。
  • 色彩特徴量も重要: 彩度の標準偏差（saturation_std）と色相の平均値（mean_hue）が上位にランクインしています。本物の絵画は芸術的訓練により色の多様性が制御されている可能性があります。
  • SHAP値の分布: 各特徴量のSHAP値は、正の値（フェイクと予測）と負の値（本物と予測）の両方を持ち、値によって予測が異なることを示しています。
